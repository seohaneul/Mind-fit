# [Mind-Fit] 서비스 기획서

## 1. 서비스 개요
- **서비스명**: Mind-Fit (마인드 핏)
- **슬로건**: "내 마음과 몸의 균형을 찾는 AI 퍼스널 트레이너"
- **서비스 형태**: 반응형 웹 애플리케이션 (Mobile First Web App)
- **기획 배경**: 
  - 신체적 건강뿐만 아니라 '마음의 건강' 중요성 대두.
  - 많은 사람들이 자신의 체력 수준을 객관적으로 알지 못하며, 어떤 운동이 내 기분에 도움이 되는지 모름.
  - 공공데이터(국민체력100 등)를 활용하여 객관적인 지표를 제공하고, AI 기술을 통해 개인화된 경험을 제공하고자 함.

---

## 2. 타겟 사용자 (Persona)
1. **2030 직장인**: 업무 스트레스로 인해 멘탈 관리가 필요하며, 효율적인 운동을 찾고 싶은 사람.
2. **건강 관리 입문자**: 내 체력이 어느 정도 수준인지 궁금하고, 집 근처에서 저렴하게 이용 가능한 공공 체육시설 정보를 원하는 사람.
3. **감정 기복이 심한 사용자**: 그날그날 기분에 따라 적절한 야외 활동이나 운동을 추천받고 싶은 사람.

---

## 3. 핵심 기능 (Key Features)

### A. AI 감정 분석 및 운동 처방 (Mind Care)
- **기능 설명**: 사용자가 선택한 기분/스트레스 지수와 직접 작성한 '감정 일기'를 AI가 분석.
- **기술 활용**: Google Gemini API (생성형 AI).
- **사용자 경험**:
  - *"일이 너무 많아서 답답해"* 라고 입력하면, AI가 심리 상담가처럼 공감해주며 스트레스 해소에 좋은 '고강도 유산소' 또는 '명상'을 추천.

### B. 신체 데이터 기록 및 비교 분석 (Physical Care)
- **기능 설명**: 사용자의 신체 지표(악력, 유연성, BMI 등)를 입력받아 저장하고 시각화.
- **데이터 활용**: 국민체육진흥공단(KSPO) '국민체력100' 연령별/성별 평균 데이터.
- **시각화**: 나의 기록 vs 대한민국 평균 데이터를 그래프(Bar Chart)로 비교하여 객관적 수준 파악.
- **히스토리 관리**: 과거 기록을 누적하여 체력 변화 추이를 확인 가능.

### C. 맞춤형 공공시설 및 프로그램 추천 (Location & Facilities)
- **기능 설명**: 사용자의 **현재 기분**과 **위치(GPS)**를 기반으로 최적의 공공 체육시설 추천.
- **상세 기능**:
  - 우울할 땐 '산책로'나 '수영장', 스트레스가 많을 땐 '테니스장' 등 키워드 기반 매칭.
  - **시설 팝업**: 시설 클릭 시 해당 시설에서 운영 중인 **체육 프로그램(강좌, 교실 등)** 상세 정보 제공.
- **데이터 활용**: KSPO 공공 체육시설 및 강좌 정보 데이터 (CSV/API).

### D. 사용자 관리 (Membership)
- **회원가입/로그인**: 이메일 기반, 연령/성별 정보 수집 (통계 비교용).
- **마이페이지**: 개인 정보 수정, **데이터 초기화(Reset)**, 회원 탈퇴 기능 제공.

## 4. 서비스 아키텍처 및 기술 스택

### Frontend
- **Framework**: React (Vite)
- **Language**: JavaScript (ES6+)
- **Styling**: Tailwind CSS (모바일 반응형 디자인 최적화)
- **Visualization**: Recharts (데이터 차트 구현)
- **State Management**: Context / Local State (localStorage 활용)

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: MongoDB (Atlas) - 사용자 정보 및 로그 저장.
- **API Integration**: Google Gemini (AI), KSPO Public Data Integration.

### Deployment
- **Frontend**: Vercel
- **Backend**: Render, Railway 또는 Vercel (Serverless)

## 5. 사용자 시나리오 (User Flow)

1. **접속 및 로그인**:
   - 메인 랜딩 페이지에서 서비스 소개 확인.
   - 로그인 또는 회원가입 진행 (비회원도 일부 기능 체험 가능).

2. **오늘의 상태 기록**:
   - 현재 기분(5단계), 스트레스(5단계), 감정 노트 입력.
   - AI가 즉석에서 위로의 말과 추천 운동 키워드 제시 ("오늘은 가벼운 조깅 어때요?").

3. **결과 확인 및 시설 탐색**:
   - AI 분석 결과 카드 확인.
   - 내 주변(GPS) 추천 시설 리스트 확인 (TOP 6).
   - 시설 클릭 -> 운영 중인 프로그램 정보 팝업 확인.

4. **신체 기록 및 통계 (내 기록 페이지)**:
   - 본인의 운동 기록 입력 (악력, 유연성 등).
   - 그래프를 통해 "나는 30대 남성 상위 20% 수준이구나" 등 인사이트 획득.

---

## 6. 데이터베이스 설계 (Schema)
- **Users**: 사용자 기본 정보 (ID, PW, 이름, 나이, 성별).
- **MentalLogs**: 날짜별 기분, 스트레스, 일기, AI 처방 기록.
- **PhysicalLogs**: 날짜별 신체 측정 수치 (악력, 윗몸 등).
- **Stats (Reference)**: KSPO 공공 데이터 (비교용 기준값).

---

## 7. 향후 발전 방향 (Roadmap)
1. **커뮤니티 기능**: 비슷한 체력/고민을 가진 사람들끼리 소통 (오운완 인증 등).
2. **웨어러블 연동**: 스마트워치 데이터(심박수, 수면) 자동 연동.
3. **전문가 매칭**: AI 추천을 넘어 실제 주변 트레이너/상담사 연결 서비스.
